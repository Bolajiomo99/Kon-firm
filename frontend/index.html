How a payment gets confirmed

The browser never decides what was paid.

1
Cart is priced server-side The client sends product IDs; the server sets the price.
2
Monnify initialises the transaction We receive a checkout URL, valid for 40 minutes.
3
Customer pays on Monnify Card details never touch our servers.
4
Signed webhook settles the order HMAC-SHA512 verified. Replays are ignored, never double-counted.
